1. Сортировка результатов поиска по возрастанию цены (test_search_results_sorting)

Шаги:
 1. Перейти на главный экран
 2. Нажать на строку поиска
 3. Ввести в поиск запрос, по которому результаты поиска не будут пустыми. Например, “Gucci”
 4. Выполнить поиск
 5. Нажать кнопку выбора способа сортировки результатов поиска
 6. В появившемся списке выбрать сортировку по убыванию цены

Ожидаемый результат:
 - Результаты поиска отсортированы по убыванию цены
